Our Team

We have developed the our-team component for admint.io/about, which matches the design currently displayed on the live site.

How to use?

To use the our-team component you need to import lib.ftd & common inside your .ftd file. The our-team component uses members-list which is defined inside common/index.ftd file.

In the common/index.ftd we have defined a list named members of record team which generates members list using title, designation, avatarandlinkattributes ofteamrecord. You can add or update thismemberslist via editingcommon/index.ftd` file.

import the lib and common dependencies inside your .ftd file as mentioned below, and you can simply utilize it.
-- import: admint.io/lib
-- import: admint.io/common
Lang:
ftd
Once you import the lib inside your .ftd file, use the our-team component by adding all required attributes of it.
Input
-- lib.our-team:

-- lib.our-team.title:

-- ftd.container:

-- ftd.text: Our
display: inline
margin-right.px: 11

-- ftd.text: Team.
color: $inherited.colors.cta-primary.base
display: inline

-- end: ftd.container

-- end: lib.our-team.title

-- end: lib.our-team

-- lib.our-team.members-list: 

-- lib.member-feature-card: **Harish Shankaran**
designation: Founder
avatar: $assets.files.static.about-us.hareesh.jpeg
link: https://www.linkedin.com/in/harishshankaran/

-- lib.member-feature-card: **Jaylah Consejo**
designation: Founder
avatar: $assets.files.static.about-us.Jaylah-Consejo.jpeg
link: https://www.linkedin.com/in/jaylahconsejo/

-- end: lib.our-team.members-list
Lang:
ftd
